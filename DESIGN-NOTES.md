# Kazyna Economics — Minimal Design

## Overview
Clean, minimal economics education platform with Kyrgyz ethno accents.

## Navigation (Header)
- **Home** — Landing page with 3 tabs
- **Microeconomics** — Micro topics
- **Macroeconomics** — Macro topics  
- **Olympiad Prep** — Competition preparation

## Three Main Tabs (Homepage)
1. **Economics for Olympiad Preparation** — 3-step roadmap
2. **Microeconomics** — 9 core topics
3. **Macroeconomics** — 8 core topics

## Removed
- Tests page (from nav)
- Glossary page (from nav)
- Search button
- Language toggle
- Articles section
- Newsletter section
- Stats cards

## Design Principles
- **Minimal** — Clean backgrounds, subtle shadows
- **Comfortable** — Soft colors, good spacing
- **User-friendly** — Clear hierarchy, 3 main sections
- **Kyrgyz accents** — Subtle top stripe, minimal ornaments

## Color Palette
- Primary: `#c8392b` (Kyrgyz red)
- Accent: `#d4a017` (Gold)
- Background: `#fafaf8` (Warm white)
- Text: `#1a1a1a` (Soft black)

## Typography
- Headings: Playfair Display (serif)
- Body: Manrope (sans-serif)

## Key Features
- Tab navigation for 3 main sections
- Topic cards with clear numbering
- Minimal ornamental accents
- Clean footer with essential links only
